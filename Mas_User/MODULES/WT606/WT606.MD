# Module_WT606 陀螺仪模块文档

## 1. 概述

`module_wt606` 是针对维特智能WT606 陀螺仪/惯性测量单元（IMU）编写的驱动模块。该模块通过 UART 进行数据交互，波特率为 92160*，能够实时获取高精度的加速度、角速度及欧拉角数据。

---
  
## 2. 架构设计

### 2.1 数据结构

### Module_WT606_Device_t - 维特陀螺仪实例

```c
typedef struct {
    uint8_t        initialized;      // 模块初始化标志
    float          acc[3];           // 加速度 (m/s²) [x, y, z]
    float          gyro_dps[3];      // 角速度 (°/s)
    float          gyro_rps[3];      // 角速度 (rad/s)
    float          Euler_degree[3];  // 欧拉角 (°) [roll, pitch, yaw]
    float          Euler_rad[3];     // 欧拉角 (rad)
    float          YawTotalAngle_degree; // 累计 Yaw 角度 (°)
    int32_t        YawRoundCount;    // Yaw 轴过圈计数
    UART_Device   *uart_dev;         // 底层 UART 设备句柄
    OfflineDevice *offline_dev;      // 离线监测句柄
} Module_WT606_Device_t;

```
### 2.2代码框架
 ```c
内部辅助函数层：
└─ DWT606_Parse()           //WT606数据读取校验

公开 API 层：
├─ Module_WT606_init()              // 模块初始化
├─ Module_WT606_task_function()     // 读取 UART 缓冲区数据
└─ Module_WT606_get_device()        // 获取陀螺仪数据的外部接口
```


### 2.3 通信协议

WT606 采用 11 字节的定长数据帧进行通信：

* 帧头：`0x55`
* 类型 (Type)：
* `0x51`：加速度、温度帧
* `0x52`：角速度、电压帧
* `0x53`：欧拉角、版本帧


* 校验位：前 10 字节累加和的低 8 位。

## 3. 核心功能实现

### 3.1 协议解析 (`DWT606_Parse`)

寻找帧头 `0x55`，并通过校验和验证数据的有效性。针对 Yaw 轴，模块实现了过圈计数逻辑，解决了角度在边界跳转导致的控制突变问题。

### 3.2 平台适配

模块根据 MCU 型号自动选择 UART 端口：

* **STM32H723xx**：使用 `huart10`
* **STM32F407xx**：使用 `huart1`

---

## 4. API 接口说明

```c
UINT Module_WT606_init(void);`
```
* **功能**：配置串口波特率为 921600，初始化 DMA 接收缓冲区，并注册离线检测节点。
* **返回值**：`TX_SUCCESS` 表示成功。

```c
void Module_WT606_task_function(void);`
```
* **功能**：读取 UART 缓冲区数据。该函数为阻塞式读取（`TX_WAIT_FOREVER`），建议在独立的陀螺仪任务线程中调用。

```c
Module_WT606_Device_t *Module_WT606_get_device(void);`
```
* **功能**：获取陀螺仪数据的外部接口。

---
## 5. 注意事项

- 内存对齐：`data_buffer` 使用了 `__attribute__((aligned(32)))`，这是为了兼容 H7 系列 MCU 的 D-Cache 一致性要求，防止 DMA 传输数据与 CPU 缓存冲突。
