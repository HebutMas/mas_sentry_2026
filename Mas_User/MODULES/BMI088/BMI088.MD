# 模块层 BMI088 驱动文档

## 1. 概述

BMI088 是一款高性能 6 轴惯性测量单元（IMU），包含一个 16 位加速度计和一个 16 位陀螺仪。本驱动模块支持 STM32H7 和 STM32F4 系列，集成了传感器初始化、数据读取、自动校准以及基于 PID 的恒温控制功能。

---

## 2. 架构设计

### 2.1 数据结构

#### BMI088_Cali_Offset_t - 校准参数

```c
typedef struct
{
    // 标定数据
    float   AccelScale;
```

#### Bmi088_device_t - 设备实例

```c
typedef struct {
    SPI_Device     *acc_device;       // 加速度计 SPI 设备句柄
    SPI_Device     *gyro_device;      // 陀螺仪 SPI 设备句柄
    PWM_Device     *bmi088_pwm;       // 加热电阻 PWM 句柄
    PIDInstance     pid_temp;         // 温度控制 PID 实例
    
    float           acc[3];           // 解析后的加速度数据 (m/s^2)
    float           gyro[3];          // 解析后的陀螺仪数据 (rad/s)
    float           temperature;      // 当前传感器温度 (℃)
    
    BMI088_Cali_Offset_t BMI088_Cali_Offset; // 校准参数
    uint8_t         initialized;      // 初始化完成标志
    uint32_t        BMI088_ERORR_CODE;// 错误状态码
} Bmi088_device_t;

```

## 3. 控制原理与功能

### 3.1 模式切换

BMI088 加速度计上电后默认进入 I2C 模式。驱动通过一次“伪读取”（Fake Read）产生 SPI 片选信号上升沿，强制芯片切换至 SPI 通讯模式。

### 3.2 恒温控制

为了减少由于环境温度变化引起的传感器漂移，驱动通过 PWM 控制加热电阻：

* 反馈量：传感器内部温度计读数。
* 控制算*：位置式 PID。
* 执行器：TIM 输出 PWM 控制加热片功率。

### 3.3 自动校准与存储

* 校准：模块支持计算陀螺仪零偏和加速度计比例因子。
* H7 系列：使用外部 W25Q64 闪存存储校准值。
* F4 系列：使用片内 Flash 存储。

## 5. 注意事项

1. Dummy Byte 处理：
* 加速度计（Accel）读取寄存器时需要 2 个 dummy bytes。
* 陀螺仪（Gyro）读取时需要 1 个 dummy byte。
* 驱动已在 `_bmi088_readdata` 中自动处理。
2. 加热安全：`BMI088_TEMP_SET` 设定不宜过高，避免损坏传感器或烫伤外壳。

## 6. 使用示例

```c
// 1. 系统初始化

void App_Ins_Init(void)
{
    Module_BMI088_init();
}

// 2. 定时任务
void Ins_Task(void) {
    // 获取原始数据
    Module_BMI088_get_accel();
    Module_BMI088_get_gyro();
    Module_BMI088_get_temp();
    
    // 恒温控制
    Module_BMI088_temp_ctrl();
    
}

// 3. 获取IMU设备
void App_gimbal_Init(void)
{
    // 获取设备实例并使用数据
    Bmi088_device_t *imu = Module_BMI088_get_device();
    float roll_speed = imu->gyro[0]; //此数据在pid/lqr初始化中进行反馈数据指针指向

}

```
